# 데이터로 본 2026 월드컵

FIFA 랭킹 기반 **Elo**로 32강부터 결승까지 **50만 회 몬테카를로 시뮬레이션**을 돌려 매일 예측을 발행하고, 대회가 끝난 뒤 얼마나 맞았는지를 지도로 되돌아보는 페이지.

🔗 https://dev.byb.kr/worldcup-2026

## 이 폴더에 대해

`index.html` 한 장이 전부인 **최종 정적 스냅샷**입니다.

- 대회 기간(2026년 6~7월)에는 별도 작업 저장소에서 매일 시뮬레이션을 돌려 새 버전을 발행했습니다. 발행 파이프라인·시뮬레이션 코드(Python)는 이 폴더에 포함하지 않고, 결과물(예측 데이터가 박힌 HTML)만 옮겨왔습니다.
- 대회 종료로 **실시간 응원 보드(구글 스프레드시트/Apps Script) 연동은 제거**했고, 그동안의 응원은 하드코딩으로 남겨 읽기 전용으로 보여줍니다.
- 페이지 안 `VERSIONS` 배열에 6/24부터 7/20까지 매일의 예측이 그대로 담겨 있어, '총정리' 탭에서 발행별 적중 지도를 볼 수 있습니다.

## 결과 한 줄 요약

🏆 우승 스페인 · 🥈 아르헨티나 · 🥉 잉글랜드 · 4위 프랑스. 우리 모델의 전체 예측 적중률은 **약 62%**였고, 정작 마지막 결승(스페인 우승)은 끝내 못 맞혔습니다.
