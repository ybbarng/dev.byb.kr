# 못생긴 리모컨 (ABC · Alexander Bonaparte Cust)

> 2018~2022, Home Assistant를 쓰기 전에 **직접 만들어 쓰던 자체 스마트홈**의 웹 리모컨.
> 이 폴더는 그 UI의 **아카이브 데모**입니다(실제 기기와 연결되지 않음). → https://dev.byb.kr/ugly-remote

## 무엇이었나
집 안 세 기기를 **사람 이름**으로 부르며 제어한 개인 홈 인프라:
- **Audrey** — 에어컨. 아두이노 IR 블래스터가 LG 리모컨 신호를 복사·재생.
- **Brice** — 조명. 상용 BLE "Switcher"를 리버스엔지니어링(방등·복도등).
- **Cathy** — 공기질. Awair Mint(사설 API), 종합점수·온습도·VOC·초미세먼지.

제어 수단 셋: 이 **웹 리모컨**, **Slack 슬래시 커맨드**, **cron 자동화**.

## 이름의 유래
애거사 크리스티 『ABC 살인사건』의 소심한 인물 **"Mr. Alexander Bonaparte Cust"** 에서 땄습니다. 리모컨은 스스로 *"못생긴 리모컨"* 이라 자백하고, 마스코트가 이렇게 사과합니다:

> "흠..이건 좀 못생겼는데, 당분간은 어쩔 수가 없군요. 일단 쓰고 계시면 더 멋있는 녀석으로 준비해서 드리겠습니다."

그 *"더 멋있는 녀석"* 은 결국 8년 뒤 **Home Assistant**가 되었습니다.

## 데모 안내
- **실제 기기와 연결되지 않습니다.** 버튼을 누르면 당시 Slack 응답 톤의 토스트가, Cathy는 샘플 공기질이 표시됩니다.
- 원본은 Flask 템플릿 `abc/abcust/templates/remote-controller.html`. 여기서는 `fetch`(`/abc/api/*`)를 스텁 처리하고 정적 에셋만 상대경로로 옮겼습니다. CSS·마스코트·사과문은 2018년 원본 그대로입니다.
